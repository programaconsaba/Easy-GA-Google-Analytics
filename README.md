# Easy GA Google Analytics - OPEN SOURCE 

Plugin de WordPress que añade Analítica en todas las páginas.

Simplemente copia y pega el código de seguimiento de Google Analytics añadiendo así analítica a todas las páginas.

## Funcionalidades

* Analítica en todas las páginas.
* No se añade analítica a las páginas si el usuario es Administrador.

# Easy GA Google Analytics - OPEN SOURCE 

WordPress plugin that enables google analytics on all pages.

Copy&Paste your Google Analytics code snippet and enables google analytics on all pages.

## Features

* Google analytics on all pages.
* Disabled analytics if the user is an Administrator.
